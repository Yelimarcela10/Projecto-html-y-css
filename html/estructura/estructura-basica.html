    Estructura básica de un documento HTML

    (Lenguaje de Marcado de Hipertexto) Es el lenguaje estándar utilizado para crear y estructurar contenido en la web. HTML es el lenguaje fundamental que se utiliza para diseñar y organizar páginas web. Funciona mediante el uso de etiquetas o elementos que indican al navegador web cómo presentar el contenido en una página.

    Etiquetas usadas para la estructura básica

    Las etiquetas más utlizadas para mantener una correcta estructura de una página web:

    Etiqueta Descripción
    <!DOCTYPE> Determina el tipo de documento HTMl en el que trabajamos.
    <html> Con esta etiqueta el documento se definirá como un HTML
    <head> En esta etiqueta se almacenan los metadatos necesarios para el documento.
    <title> Contiene el título del documento
    <body> En esta etiqueta se almacena todo el contenido de nuestra página, tanto como contenedores como imágenes.
    <section> Podemos agregar varias secciones de nuestro contenido y agregar nuestros elementos.
    <footer> Nos indica el pie de página.
